# tc9305.github.io

Personal GitHub Pages site for Shuo-Wen Chiang. This repository hosts a single-page HTML portfolio and resume-style homepage that presents academic background, coursework, projects, and extracurricular activities.

## Overview

This site is designed to be a clean personal landing page with the following sections:

- Intro and contact links: name, short profile, resume, GitHub, email, Instagram, Threads, and phone.
- Academic timeline: undergraduate and graduate study path.
- Coursework: AI, computer vision, systems, networking, and core CS classes.
- Experience and projects: deep learning, biomedical information, internship work, cloud deployment, and automation.
- Extracurricular activities: teamwork, student organization work, and personal interests.

## Live Site

If GitHub Pages is enabled for this repository, the site can be published directly from the `main` branch using the root `index.html` file.

## Tech Stack

The page is built with lightweight front-end technologies:

- HTML5 for semantic page structure.
- CSS3 for the full visual design and layout.
- CSS Grid and Flexbox for responsive cards, timelines, and button groups.
- Media queries for mobile-friendly behavior.
- Google Fonts for typography.
- Font Awesome for icons.
- Inline SVG background textures for the patterned visual style.

## Design Notes

The visual direction uses a dark theme with orange highlights, glowing accents, and card-based sections. The overall style is inspired by anime-themed branding while still functioning as a professional personal site.
