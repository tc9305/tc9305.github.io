專業經歷與學術專案 // Experience
深度學習與 AI 模型應用：生醫資訊與智慧寵物辨識系統 學術專案
  • 電腦視覺 (YOLO 寵物辨識)：自主完成自定義資料集的蒐集、影像標註與資料增強，透過 YOLO 模型調參最佳化，大幅提升即時物件偵測的推論速度與準確度。
  • 生醫資訊 (蛋白質序列預測)：處理高維度生物序列非結構化資料，進行數據前處理、特徵萃取與序列模型訓練，精準預測其潛在特性。
  • 整合 OpenCV 影像處理工具與實體 IoT 裝置，成功實現端到端 (End-to-End) 的「即時影像辨識系統」。
達方電子 (DARFON) ITS 部門 BPM 團隊 — 系統開發實習生 企業實習 (2025.01 - 2025.09)
  • 基於 ASP.NET MVC 框架，使用 C#、JavaScript、jQuery 實作與重構前後端核心系統功能。
  • 設計與維護 SQL Server 資料庫，編寫預存程序 (Stored Procedure) 與 Trigger，確保資料高度一致性。
  • 透過執行計畫進行 Query 索引優化 (Query Indexing) 與批次大數據資料處理，顯著提升系統查詢效率。
  • 開發主子表資料關聯、報表查詢與自動化資料匯出功能。
自主雲端部署與自動化通知系統 個人項目
  • 獨立建置與配置個人實體 NAS 伺服器，熟練掌握 HTTP/HTTPS/TCP 等網路底層協議。
  • 結合 Webhook 與 API 技術整合開發 Line Bot，實現自動化即時通知與系統警報功能。
