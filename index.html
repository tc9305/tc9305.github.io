我的忍道 // NINDO

「言出必行，這就是我的忍道！」
嗨，我是碩文，目前在元智大學鑽研「資工之術」（資工系四年級）。我的主要修煉領域圍繞在人工智慧應用與後端系統架構整合。我不僅熟練於資料處理、後端 API 開發，更具備雲端與 Docker 部署的經驗。對我而言，把 AI 預測模型成功整合到實體應用、解決生活周遭的具體問題，就像成功研發一門實用的新忍術一樣，無比暢快！

除了個人的硬實力，我極度在乎同伴間的「羈絆」（溝通交流與團隊合作）。在實驗室與專題修行外，我也喜歡涉足各種新奇領域。我期許自己在未來的研究所階段能如「開啟仙人模式」一般，全面深化對演算法及 AI 模型的根本性理解！

任務卷軸 // MISSIONS
生物資訊：特徵選取與預測平台 S級任務
以預測轉錄蛋白中 ATP 結合位點為目標，建構高精準度的預測系統。
  • 進行生物序列資料前處理，使用 Sliding Window 機制進行特徵提取。
  • 召喚並訓練 XGBoost 與 Random Forest 等多種機器學習演算法。
  • 以 Flask 架設後端 API，運用 Docker 進行容器化「封印」，並成功部署於 GCP 雲端平台。
達方電子 ITS 部門 BPM Team 實習生 A級任務 (2025.01 - 2025.09)
進入企業級真實戰場，深度參與內部核心系統維護與功能優化。
  • 基於 ASP.NET MVC 架構，負責企業內部系統的前後端維運與重構。
  • 交替使用 C#、JavaScript 及 jQuery 等多種印記快速編寫功能。
  • 設計 SQL Server 資料庫，編寫 Stored Procedure 讓查詢加速（效率堪比瞬身之術）。
IoT 與即時影像辨識系統：寵物小管家 B級任務
結合嵌入式軟硬體與智慧辨識，守護木葉村毛孩們的日常健康。
  • 使用 Raspberry Pi 結合 YOLO 偵測模型，實作即時動物偵測與警報功能。
  • 打通感測器資料與影像的即時傳輸機制，搭建穩固的資料庫存取後端。
